Questions Answered

What is the purpose of HTML5 semantic tags? Which ones do you see as most useful?
HTML5 Semantic tags attribute a block of text with a specific style or order. They can be used to organize documents to be ready easily by visitors, admins, or screen readers. The most useful tag is "section" as it allows for a body of text to be organized by subjects.

What is the order the browser uses to display elements on a page?
Top to bottom, left to right

What did you learn about sublime in this challenge? Do you think you'll create more handy snippets at a later date?
I learned what the exact syntax meant for creating snippets. I would use snippets if I find myself needlessly repeating code on every new file I create.

What concept in this challenge was most difficult for you? How can you improve your understanding of this concept?
Understanding the syntax for sublime snippets was most difficult. It was new subject matter that I know I will need more time on. Reading more into it and applying those principles will definitely help me understand the topic better.
